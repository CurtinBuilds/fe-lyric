# xhs-assets

小红书短视频相关物料统一放这里。

建议目录：

```text
xhs-assets/
├── audio/
│   ├── ambience/
│   ├── effects/
│   └── music/
├── fonts/
├── svg/
│   ├── birds/
│   └── ornaments/
└── textures/
```

用途说明：

- `audio/ambience/`：雨声、风声、沙声、瀑布环境音
- `audio/effects/`：钟声、印章声、鸟鸣等点状音效
- `audio/music/`：背景乐
- `fonts/`：楷体、题签字体
- `svg/birds/`：白鹭、大雁等飞鸟素材
- `svg/ornaments/`：题签、印章、云纹、卷轴边饰
- `textures/`：半透明纸面、暖纸色底纹、UI 纹理
